Applications web de messagerie

Dans le cadre de mes études, je suis actuellement en train de travailler sur la conception d'une application de messagerie Web. Du wireframe, de la base de données, à la conception de l’application. C'est un projet sans utilisation de frameworks.

Technologies utilisées :

  • HTML/CSS
  • JavaScript
  • PHP
  • SQL
  • Requêtes fetch JSON
  • JMerise (Pour la création du MCD)
  • Déploiement du serveur sur OVH
  • NGINX
  • Protection contre les injections SQL
  • Protection contre les failles XSS
  • Versioning : Git
  • Outil de gestion de projet : Trello
  • Conception de la maquette sur Canva et Figma
  • Définition d'une charte graphique
  • Site Responsive
